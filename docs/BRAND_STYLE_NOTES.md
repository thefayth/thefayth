# Brand Style Notes

## Brand Qualities

- bold
- intelligent
- strategic
- human
- memorable
- archival
- world-building
- privacy-aware

## Visual Direction

The selected project visuals combine ethereal archive imagery, interface cards, mobile/desktop previews, and the line `memory, signal, world-building.`

## Color Direction

- deep plum and night tones
- rose and copper highlights
- soft luminous archive textures
- black/green Matrix-inspired live-site option for technical surfaces

## Typography Direction

- editorial serif for public storytelling
- clean readable text for body copy
- monospace green/black styling only when used deliberately for live technical atmosphere

## Avoid

- generic SaaS templates
- stock startup dashboards
- fake product claims
- screenshots of private systems
- anything that implies open-source licensing
