# THEFAYTH Project Brief

THEFAYTH is a public-facing memory, archive, forum, and project-gateway surface connected to Faith Cheltenham's broader ecosystem.

It presents a protected public layer for:

- memory rooms and discussion threads
- archive and project navigation
- multilingual retellings
- AI-assisted but human-reviewed forum work
- source-backed public storytelling
- ownership-aware project presentation

The private engine remains outside this repository.

## Core Idea

Public surface. Private engine. Receipts always.

## Audience

- Visitors trying to understand THEFAYTH.
- Supporters and collaborators who need a public overview.
- Readers moving between thefayth.net and FaithCheltenham.com.
- Future partners who need to respect the ownership boundary before asking for access.

## Why Now

THEFAYTH has grown from static supersite and concept visuals into a live WordPress/forum surface. This repository gives the project a professional GitHub presence without releasing private systems.
