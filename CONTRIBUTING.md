# Contributing

This repository is not accepting public source contributions.

Public feedback may be considered when it respects the project boundary:

- No requests for private source code, credentials, prompts, or operational access.
- No attempts to extract unpublished creative work or private workflows.
- No commercial use proposals without a clear written agreement.
- No redistribution or training-use assumptions.

For collaboration, begin with the public project materials and contact Faith through FaithCheltenham.com.
