# Security Policy

Do not submit secrets, credentials, private prompts, unpublished manuscripts, family information, legal/admin material, medical material, benefits records, customer data, or operational infrastructure details through public GitHub issues or pull requests.

For security, privacy, or provenance concerns, use the contact path on FaithCheltenham.com or the public contact route connected to THEFAYTH.

This repository intentionally excludes the private engine and deployment system.
